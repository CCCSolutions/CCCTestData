# CCC Test Data

Raw and archival test data for Canadian Computing Competition (CCC) problems.

## Structure

- `raw-2003-and-below/` - Original test data for 1996-2003 CCC problems (semantic filenames)
- `raw-2016-plus/` - Original test data for 2016-2025 CCC problems (Junior + Senior)

Normalized test data lives in the main [CCCSolutions](https://github.com/CCCSolutions/CCCSolutions) repo under `website/public/past_contests/{year}/{problem}/test_data/`.

**Note:** some test cases missing from the main repository may be located here.
